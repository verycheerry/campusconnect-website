# CampusConnect CI/CD Project 🚀

## 📌 Project Overview
This project demonstrates a CI/CD pipeline for deploying a static website using GitHub Actions and AWS EC2.

## ⚙️ Technologies Used
- GitHub
- GitHub Actions (CI/CD)
- AWS EC2 (Ubuntu)
- Nginx Web Server
- HTML, CSS

## 🔄 CI/CD Workflow
1. Code is pushed to GitHub
2. GitHub Actions is triggered
3. Files are copied to EC2 server
4. Nginx serves the updated website

## 🌐 Website Features
- Home Page
- Contact Page
- Basic styling

## 🚀 Deployment
The website is automatically deployed to EC2 after every push to the main branch.
